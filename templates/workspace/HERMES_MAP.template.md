# Hermes Map

Copy this file to `HERMES_MAP.md` and fill in local paths.

## Hermes

- Hermes home: `%LOCALAPPDATA%\hermes`
- Hermes agent repo: `%LOCALAPPDATA%\hermes\hermes-agent`
- Hermes CLI: `%LOCALAPPDATA%\hermes\hermes-agent\venv\Scripts\hermes.exe`
- Config: `%LOCALAPPDATA%\hermes\config.yaml`
- Secrets: `%LOCALAPPDATA%\hermes\.env`
- Auth store: `%LOCALAPPDATA%\hermes\auth.json`

Do not symlink or expose `.env` or `auth.json` into the workspace.
